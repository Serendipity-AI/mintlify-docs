## Quick start

```bash
curl -X POST https://api.markdown2pdf.ai/v1/convert \
     -H "Content-Type: text/markdown" \
     -H "Authorization: L402 <macaroon>:<preimage>" \
     --data-binary @your_file.md \
     --output your_file.pdf
```
